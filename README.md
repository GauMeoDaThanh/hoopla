### How to run project - Ubuntu/WSL

1. `curl -LsSf https://astral.sh/uv/install.sh | sh`
2. `git clone https://github.com/GauMeoDaThanh/hoopla.git`
3. `cd hoopla`
4. `source .venv/bin/activate`
5. `uv run cli/keyword_search_cli.py`
